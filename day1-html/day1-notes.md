# Day 1 Notes

## Self-test Answers

Q1: Semantic HTML — Elements that convey meaning (e.g., header, article); improves accessibility and SEO.

Q2: section vs div — `section` is a semantic, thematic grouping (usually with a heading); `div` is a generic container for layout or styling.

Q3: alt purpose — Provides descriptive text for images used by screen readers and shown when images fail to load.

Q4: viewport meta — Tells the browser to size the layout to the device width and set the initial zoom for responsive rendering.

Q5: button vs a — `button` performs actions (submit, JS); `a` navigates to URLs; use each for its semantic purpose.
